# Tailwind Color Configuration

## Color Choices

- **Primary:** `blue` — Used for buttons, links, key accents
- **Secondary:** `sky` — Used for tags, highlights, secondary elements
- **Neutral:** `neutral` — Used for backgrounds, text, borders

## Usage Examples

Primary button: `bg-blue-600 hover:bg-blue-700 text-white`
Secondary badge: `bg-sky-100 text-sky-800`
Neutral text: `text-neutral-600 dark:text-neutral-400`

## Tailwind Configuration

If using Tailwind CSS, these colors are available by default. No custom configuration needed — use the standard Tailwind color utilities:

- `blue-*` for primary actions
- `sky-*` for secondary elements
- `neutral-*` for backgrounds, text, and borders
